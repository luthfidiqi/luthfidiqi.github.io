## **Installation**

- Clone Repositories

```
https://github.com/luthfidiqi/luthfidiqi.github.io
```

- Open Project Folder

```
$ cd luthfidiqi.github.io-master
```

- Intall NPM Package

```
npm install
```

- Start The Project

```
npm start
```
